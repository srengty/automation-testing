## 🔬 Lab Exercises

### **Lab 1: Calculator Testing (20 minutes - Guided)**

**Blackbox Focus:**
- Equivalence partitioning for arithmetic operations
- Boundary value analysis for edge cases
- Error condition testing (division by zero, invalid inputs)

**Whitebox Focus:**  
- Statement coverage measurement
- Branch coverage for conditional logic
- Path coverage for complex methods (power, factorial)

### **Lab 2: BankAccount Testing (25 minutes - Independent)**

**Complex Business Logic:**
- Account types and validation rules
- Transaction fees and overdraft protection  
- Daily limits and fraud detection
- Auto-upgrade and maintenance fees

**Advanced Testing Techniques:**
- State-based testing for account status
- Complex condition coverage
- Integration scenarios (transfers)
- Error recovery testing
